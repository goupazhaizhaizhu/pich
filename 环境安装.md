# 环境安装

## mysql

### 启动mysql

| linux                       | mac                     |
| --------------------------- | ----------------------- |
| sudo systemctl start mysqld | sudo mysql.server start |

## redis

### 安装redis

```bash
linux: 
sudo yum install epel-release
sudo yum install redis
```

### 启动redis

```bash
linux:
sudo systemctl start redis

mac: 
brew services start redis
```

### 启动命令行

```bash
linux:
redis-cli
```
